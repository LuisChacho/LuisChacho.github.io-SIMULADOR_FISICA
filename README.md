# Simulador de Física - 100 Reactivos Únicos

Plataforma interactiva diseñada para la evaluación diagnóstica de aspirantes universitarios en el área de Ciencias Exactas.

## Características Técnicas:
- **100 Preguntas de Física:** Ejercicios reales con datos variables sin repeticiones de Cinemática, Dinámica y Energía.
- **Navegación Libre:** Panel matricial interactivo integrado en el lado izquierdo para alternar libremente entre los reactivos del 1 al 100.
- **Sin Patrón de Respuestas:** Algoritmo Fisher-Yates integrado para mezclar el orden de las opciones (A, B, C, D) en cada carga.
- **Comodines Integrados:** 50/50, Descarte, Llamada al Tutor y Estadísticas de Grupo totalmente automatizadas con lógica contextual.
